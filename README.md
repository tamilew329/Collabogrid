A lightweight, browser-based collaborative whiteboard designed for **interactive sessions, workshops, and classrooms**.

Participants respond to prompts and see their ideas appear as cards on a shared board, creating a live, visual space for discussion and reflection.


---

## 🌐 Live Demo

👉 https://tamilew329.github.io/Collabogrid

*(No installation required — open in your browser)*

---

## ✨ Features

- 🧩 Prompt-based entry (responses become cards)
- 🖼️ Interactive whiteboard with drag & drop
- 📊 Configurable grid layout
- 💬 Comments per card
- 😊 Emoji reactions
- 📌 Sticky notes attached to cards
- ✍️ Free text annotations
- 🔗 Link & image sharing
- 📎 File uploads (session-based)
- 🔄 Real-time sync (Supabase)

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript  
- **Realtime:** Supabase  
- **Storage:** localStorage + IndexedDB  
- **Hosting:** GitHub Pages / Netlify  

---

## 🚀 Getting Started

### Run locally

1. Clone the repository  
2. Open `index.html` in your browser  

No build step required.

---

## 🌍 Deployment

You can deploy this app easily on:

- GitHub Pages  
- Netlify (recommended for future development)  

---

## 🔑 Access

The app uses simple access codes:

- Admin code → configure the board  
- User code → join the session  

⚠️ These are **client-side only** and not secure for production use.

---

## ⚠️ Limitations

- No authentication system  
- Access control is not secure  
- Supabase security depends on database rules  
- File uploads are session-based (IndexedDB)
